# BDC Design Rhythm v3.0
## Enterprise AI · Academic Intelligence · Precision-First

Welcome to the Big Data Club (BDC) Design System documentation. This system is built to provide a cohesive, premium, and academic experience across all BDC platforms.

### Core Philosophy

> **Minimal Structure. Massive Clarity. AI-Native Identity. Enterprise Precision.**

We prioritize a "Data-First" approach where content is the protagonist, supported by a clean, academic structure and subtle AI-native visual cues.

### Key Pillars

- **Academic & Minimal**: Clean layouts, avoiding unnecessary clutter or loud gradients.
- **Data-First**: Clear typography and generous spacing to ensure information density without sacrifice.
- **AI-Native**: Visual elements like pipeline steps, glow effects, and cyan accents that signal intelligence.

### Getting Started

This Storybook serves as the living documentation for our component library. You can find:

- **Foundations**: Colors, Typography, and Spacing.
- **Components**: Primitive and complex UI elements.
- **Guidelines**: Implementation details and anti-patterns.

### Design Mode

BDC platforms support both **Light** and **Dark** modes. Dark mode is our primary identity, especially for LMS and Dashboard areas, using a specific **Blue-Navy** palette.

- **Dark Mode**: High contrast, layered navy backgrounds (`#050B18` to `#132240`).
- **Light Mode**: Premium enterprise feel, using a clean slate palette.
